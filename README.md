# Mobile Tests

Автотесты для мобильных приложений VK Video и Алхимия: Головоломка с использованием Appium, Selenide и TestNG.

## Стек технологий

- **Java 22**
- **Appium 9.3.0**
- **Selenide 7.7.3**
- **TestNG 7.10.2**
- **Maven**
- **Android 11 (API 30)**
- **Appium Inspector**
## Запуск тестов
1. Запустить appium, android-emulator
2. Запустить maven
```bash
mvn clean test
```
